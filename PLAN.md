# PetPath — план разработки

## Стек
- Frontend: React + TypeScript → GitHub Pages (auto-deploy на push)
- Storage (free): Telegram CloudStorage / localStorage fallback
- Storage (premium): Supabase (подключить позже)
- AI: Gemini 2.0 Flash (бесплатный)
- Backend: Supabase Edge Functions (подключить позже, для проксирования AI ключа)
